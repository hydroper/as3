# Language specification changes

## Lexical structure

- Added punctuators:
  - `**`
  - `**=`
  - `?.`
  - `??`
  - `??=`
  - `=>`
- Added context keywords:
  - `enum`
  - `not`
  - `readonly`
  - `type`
- Added literals:
  - Triple string literal
- Added escapes:
  - Unicode scalar escape (`\u{}`), present in string literals and identifiers
